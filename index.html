HTML

HTML-Hyper Text Markup Language
HTML is the Basis for each Web Page.

HTML is made up of:
-Text Content
-Markup
-Images and Video
-Links to other pages

HTML shows the structure of the web page, and can be thought of as a Tree Like Structure which is also called the "DOM"-Document Object Module.
Text Content still shows up as text within HTML, but how you format it will change the context to be important to the reader.
Markup changes the look of plain text to Bold, Italics, etc. You can also include Links.

CSS

CSS is also a language and controls the style of a page and how it looks. Within each Web Page, the HTML Structure and CSS Style code can be viewed in the Developer Tools view. CSS allows you to use Selectors to avoid repetition in code, and to create rules for formatting. By changing one item in CSS, it will change your layout desired, and avoid the need for HTML recoding-AWESOME!!! There are several Text Editors available to assist the program writer-CodePen, ScratchPad, and Sublime Text to assist with HTML, and CSS code.

Code Structure

Before Web Designer begins to think of the Web Page code, layout is the first plan of action! How should the Website look? Since a page is essentially boxes, what content would naturally look best in each box? This is called Box Positioning and includes Margins, Borders, Padding, and Content. Repeated styles and semantic elements should then be considered. Then write the HTML. Apply the styles from BIGGEST to smallest. Then take a good look at your web page and tweak what you don't care for.
